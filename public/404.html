🚀 SparkLog R2 存储代理

这个页面可以作为 R2 存储的代理，解决 CORS 问题。无需在 Cloudflare 上配置任何内容。

📋 使用方法

在环境变量中设置：

VITE_R2_PROXY_URL=https://your-domain.com/404.html

或者直接使用这个页面的 URL 作为代理端点。

🔧 测试 R2 操作

📚 工作原理

这个代理页面通过以下方式工作：

  • 无服务器：完全在浏览器中运行，无需后端
  • 直接访问：使用 AWS SDK 直接访问 R2 存储
  • CORS 友好：避免了跨域问题
  • 易于部署：只需部署到 GitHub Pages 或其他静态托管服务
